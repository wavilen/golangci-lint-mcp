# staticcheck: S1012

<instructions>
Detects code that computes elapsed time as `time.Now().Sub(start)` instead of using the idiomatic `time.Since(start)`. The `time.Since` helper is shorter, clearer, and the standard way to measure elapsed time.

Replace `time.Now().Sub(start)` with `time.Since(start)`.
</instructions>

<examples>
## Bad
```go
start := time.Now()
doWork()
elapsed := time.Now().Sub(start)
```

## Good
```go
start := time.Now()
doWork()
elapsed := time.Since(start)
```
</examples>

<patterns>
- Using time.Now().Sub() to measure elapsed time
- Computing duration by subtracting start from current time manually
- Measuring function execution time with Sub instead of Since
- Using time.Now() twice and computing the difference
</patterns>
