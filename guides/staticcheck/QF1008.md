# staticcheck: QF1008

<instructions>
Detects unnecessary type conversions where the source and target types are identical or can be inferred. This includes `string(s)` where `s` is already a string, or `int(x)` where `x` is already an int.

Remove the unnecessary type conversion.
</instructions>

<examples>
## Bad
```go
var s string = "hello"
x := string(s) // s is already string
```

## Good
```go
var s string = "hello"
x := s
```
</examples>

<patterns>
- Converting a value to its own type (no-op conversion)
- string([]byte(s)) round-trip when unnecessary
- Explicit type conversion where inference suffices
- Redundant casts in assignment contexts
</patterns>
