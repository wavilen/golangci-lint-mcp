# staticcheck: S1006

<instructions>
Detects uses of `fmt.Printf` or `fmt.Sprintf` followed by `fmt.Println` or `fmt.Print` with a string that already contains formatting. More commonly, it flags `fmt.Print(fmt.Sprintf(...))` which should just be `fmt.Printf`.

Use `fmt.Printf` directly instead of combining format-then-print calls.
</instructions>

<examples>
## Bad
```go
fmt.Println(fmt.Sprintf("hello %s", name))
```

## Good
```go
fmt.Printf("hello %s\n", name)
```
</examples>

<patterns>
- Wrapping fmt.Sprintf inside fmt.Println
- Using fmt.Print with a pre-formatted string
- Using fmt.Sprintf to build output that is immediately printed
- Combining Printf format verbs with separate Print calls
</patterns>
