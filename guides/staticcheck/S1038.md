# staticcheck: S1038

<instructions>
Detects unnecessary formatting operations where a simpler alternative exists. This includes using `fmt.Sprintf` for string concatenation or formatting when direct operations would suffice.

Use string concatenation or simpler formatting approaches.
</instructions>

<examples>
## Good
```go
msg := "Hello, " + name + "!"
```
</examples>

<patterns>
- Replace `fmt.Sprintf` for simple string concatenation with `+` operator
- Replace `%s` single-value formatting with direct string concatenation
- Replace `fmt.Sprintf` with `strings.Replace` when substitution is clearer
- Use direct concatenation instead of over-formatting with `Sprintf`
</patterns>
