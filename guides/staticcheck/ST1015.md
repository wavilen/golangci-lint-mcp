# staticcheck: ST1015

<instructions>
Detects unnecessary code patterns in function and method declarations. This includes redundant parameter names, unnecessary grouping, or patterns that obscure the code's intent.

Simplify the declaration by removing unnecessary patterns.
</instructions>

<examples>
## Bad
```go
func (s *Server) handle(s *Server, req *Request) {}
```

## Good
```go
func (s *Server) handle(req *Request) {}
```
</examples>

<patterns>
- Duplicate parameter names that shadow the receiver
- Redundant parameter declarations
- Unnecessary grouping of parameters
- Poor function signature style
</patterns>

<related>
ST1006, ST1008, S1036
