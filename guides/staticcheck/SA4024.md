# staticcheck: SA4024

<instructions>
Detects comparisons of `time.Time` values using `==` or `!=`. The `time.Time` type contains a `loc` (location) field and monotonic clock reading, so `==` comparison may return false for times that represent the same instant but differ in location or monotonic reading.

Use `t1.Equal(t2)` to compare time values for the same instant.
</instructions>

<examples>
## Bad
```go
if startTime == endTime {
    slog.Info("same time")
}
```

## Good
```go
if startTime.Equal(endTime) {
    slog.Info("same time")
}
```
</examples>

<patterns>
- `time.Time` values compared with `==` or `!=`
- Struct equality on time values ignoring location/monotonic differences
- Comparing times from different timezones
- Mocking time in tests where monotonic readings differ
</patterns>
