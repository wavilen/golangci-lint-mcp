# staticcheck: ST1010

<instructions>
Detects unnecessary string conversions where the value is already a string or can be used as one directly. This includes `string(b)` on byte slices where `string(b)` is the only operation, or converting a `fmt.Stringer` result that is already a string.

Remove the unnecessary string conversion or use the value directly.
</instructions>

<examples>
## Bad
```go
name := string(getString())
```

## Good
```go
name := getString()
```
</examples>

<patterns>
- Converting a string result to string again
- Using string() on values already of type string
- Unnecessary []byte to string conversion for comparison
- Converting fmt.Sprintf result (already string) to string
</patterns>
