# staticcheck: S1025

<instructions>
Detects hardcoded time format strings that duplicate one of Go's standard time format constants. Go uses a unique reference time (Mon Jan 2 15:04:05 MST 2006) for formatting, and the standard library provides constants for common layouts.

Use the `time` package constants like `time.RFC3339`, `time.DateOnly`, or `time.DateTime` instead of hardcoding format strings.
</instructions>

<examples>
## Bad
```go
formatted := t.Format("2006-01-02T15:04:05Z07:00")
```

## Good
```go
formatted := t.Format(time.RFC3339)
```
</examples>

<patterns>
- Hardcoding "2006-01-02" instead of time.DateOnly
- Hardcoding "2006-01-02 15:04:05" instead of time.DateTime
- Hardcoding RFC3339 format string instead of time.RFC3339
- Using custom format strings that exactly match a standard constant
</patterns>
