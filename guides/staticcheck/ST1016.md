# staticcheck: ST1016

<instructions>
Detects uses of `interface{}` where `any` (introduced in Go 1.18) is preferred. The `any` type alias is more concise and is the modern Go convention for denoting arbitrary types.

Replace `interface{}` with `any`.
</instructions>

<examples>
## Good
```go
func process(_ any) {}
```
</examples>

<patterns>
- Replace `interface{}` with `any` (Go 1.18+)
- Replace `map[string]interface{}` with `map[string]any`
- Replace `[]interface{}` with `[]any`
- Use `any` for type assertions instead of `interface{}`
</patterns>
