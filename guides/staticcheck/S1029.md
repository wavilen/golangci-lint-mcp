# staticcheck: S1029

<instructions>
Detects unnecessary formatting operations where the value is already suitable for direct use. This includes using `fmt.Sprintf` to build a string that could be constructed more simply.

Simplify by removing the formatting layer or using a direct conversion.
</instructions>

<examples>
## Bad
```go
msg := fmt.Sprintf("error: %s", err)
```

## Good
```go
msg := "error: " + err.Error()
// or better, use errors.Wrap for wrapping:
// return errors.Wrap(err, "error")
```
</examples>

<patterns>
- Using fmt.Sprintf to concatenate strings where + works
- Using fmt.Sprintf("%s", s) when s is already a string
- Formatting single values that don't need formatting
- Over-engineered string construction with Sprintf
</patterns>
