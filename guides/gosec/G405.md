# G405

<instructions>
Detects use of DES (`crypto/des`) or RC4 ciphers. Both are cryptographically broken: DES uses a 56-bit key vulnerable to brute force, and RC4 has biases that leak plaintext information.

Use AES (`crypto/aes`) with GCM mode for symmetric encryption, or ChaCha20-Poly1305 for modern authenticated encryption.
</instructions>

<examples>
## Bad
```go
import "crypto/des"

block, _ := des.NewCipher(key) // 56-bit key — trivially breakable
```

## Good
```go
import "crypto/aes"

block, err := aes.NewCipher(key) // 128/192/256-bit key
if err != nil {
    return err
}
gcm, _ := cipher.NewGCM(block)
```
</examples>

<patterns>
- `crypto/des` used for legacy encryption compatibility
- RC4 cipher used in TLS or custom encryption protocols
- Triple DES (3DES) still in use despite being deprecated
</patterns>
