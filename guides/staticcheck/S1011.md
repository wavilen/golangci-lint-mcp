# staticcheck: S1011

<instructions>
Detects `for` loops that append elements one at a time to a slice. Using a single `append` call with all elements (or variadic expansion) is more efficient and readable — it minimizes allocations.

Replace the loop with a single `append` call.
</instructions>

<examples>
## Good
```go
items = append(items, newItems...)
```
</examples>

<patterns>
- Replace loop-based individual `append` with a single `append(items, newItems...)`
- Convert `for-range` that only calls `append` into a variadic `append`
- Use spread operator `...` to append all elements at once
- Replace element-by-element slice appending with variadic `append`
</patterns>
