# staticcheck: SA4025

<instructions>
Detects integer division results assigned to float variables, such as `float64(1 / 2)` which yields `0.0` instead of `0.5`. The integer division is performed before the conversion, truncating the fractional part. Convert the operands to floats before dividing.

Convert at least one operand to a float type before the division: `float64(a) / float64(b)`.
</instructions>

<examples>
## Bad
```go
func divide(total, count int) float64 {
    return float64(total / count) // integer division, then conversion
}
```

## Good
```go
func divide(total, count int) float64 {
    return float64(total) / float64(count) // float division
}
```
</examples>

<patterns>
- Convert operands to float before dividing: `float64(a) / float64(b)` instead of `float64(a / b)`
- Fix percentage calculations: convert before dividing — `float64(hits) / float64(total) * 100`
- Fix averaging: `float64(sum) / float64(n)` instead of `float64(sum / n)`
- Convert before division, not after — to preserve fractional results
</patterns>
