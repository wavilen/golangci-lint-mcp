# staticcheck: SA1021

<instructions>
Detects use of `bytes.Buffer.Bytes` (or `String`) after subsequent read operations on the same buffer. `bytes.Buffer.Bytes` returns a slice referencing the internal buffer, which may be invalidated by later `Read` or `Reset` calls. Capture the result before reading, or use `io.ReadAll` to get a copy.
</instructions>

<examples>
## Bad
```go
var buf bytes.Buffer
buf.WriteString("hello")
data := buf.Bytes()
buf.Read(make([]byte, 3)) // modifies internal state, data may be invalid
slog.Info("data", "value", data)
```

## Good
```go
var buf bytes.Buffer
buf.WriteString("hello")
data := make([]byte, buf.Len())
copy(data, buf.Bytes())
```
</examples>

<patterns>
- Calling `buf.Bytes()` and then reading from or resetting the buffer
- Storing a reference from `Bytes()` past the lifetime of the buffer's current state
- Assuming `bytes.Buffer.Bytes` returns an independent copy
</patterns>
