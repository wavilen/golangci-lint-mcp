# G301

<instructions>
Detects directories created with overly permissive permissions (default `0777` or world-writable). Permissive directory permissions allow any local user to read, modify, or inject files into the directory.

Use restrictive permissions when creating directories. `0750` allows owner read/write/execute and group read/execute; `0700` restricts to owner only. Apply the principle of least privilege.
</instructions>

<examples>
## Bad
```go
os.MkdirAll("/var/app/data", 0777)
```

## Good
```go
os.MkdirAll("/var/app/data", 0750)
```
</examples>

<patterns>
- Use restrictive permissions like `0750` or `0700` for `os.Mkdir`/`os.MkdirAll` — avoid `0777`
- Set restrictive permissions (`0700`) on directories that contain secrets or config files
- Limit temp directory permissions to `0700` — use `os.MkdirTemp` which sets appropriate defaults
</patterns>
