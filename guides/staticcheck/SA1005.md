# staticcheck: SA1005

<instructions>
Detects invalid time format strings passed to `time.Format`. The argument must use Go's reference time `Mon Jan 2 15:04:05 MST 2006`. If the format string contains components that don't match the reference time (e.g., `YYYY` or `DD`), the output will be wrong silently.

Replace non-standard format tokens with Go reference time components.
</instructions>

<examples>
## Bad
```go
// YYYY and DD are not Go format tokens — they output as literal text
formatted := time.Now().Format("YYYY-MM-DD")
```

## Good
```go
formatted := time.Now().Format("2006-01-02")
```
</examples>

<patterns>
- Using `YYYY` instead of `2006` for year
- Using `MM` or `DD` instead of `01` and `02`
- Mixing strftime conventions into Go's `Format` calls
- Copy-pasting format strings from other languages (Python, C)
</patterns>
