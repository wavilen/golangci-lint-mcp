# staticcheck: SA4021

<instructions>
Detects assignments of the form `x = x + y` or `x = x * y` that should use the compound assignment operators `+=` or `*=`. While functionally equivalent, compound assignment is more idiomatic, concise, and clearly communicates intent.

Replace with the compound assignment operator: `x += y` instead of `x = x + y`.
</instructions>

<examples>
## Good
```go
total += amount
name += suffix
```
</examples>

<patterns>
- Replace `x = x + y` with `x += y`
- Replace `x = x - y` with `x -= y`
- Replace `x = x * y` with `x *= y`
- Replace `x = x / y` with `x /= y`
</patterns>
