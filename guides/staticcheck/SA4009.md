# staticcheck: SA4009

<instructions>
Detects function arguments that are overwritten before being used within the function body. The parameter value passed by the caller is immediately replaced, making the argument useless. This typically indicates dead code or a parameter that should be removed.

Remove the parameter from the function signature if it is not needed, or fix the logic to use the passed-in value before overwriting it.
</instructions>

<examples>
## Bad
```go
func process(data []byte, offset int) int {
    offset = 0 // overwrites argument
    return int(data[offset])
}
```

## Good
```go
func process(data []byte, offset int) int {
    return int(data[offset])
}
```
</examples>

<patterns>
- Remove or use parameter values before overwriting them at the top of a function
- Check error parameters before overwriting them with new values
- Replace always-overriding default-value patterns with proper default handling
- Remove unused parameters from the function signature
</patterns>
