# staticcheck: ST1019

<instructions>
Detects unnecessary imports that are either unused or could be replaced by a more specific import. This includes importing packages that provide overlapping functionality.

Remove unused imports or use more appropriate packages.
</instructions>

<examples>
## Bad
```go
import (
    "io/ioutil" // deprecated since Go 1.16
)
```

## Good
```go
import (
    "io"
    "os"
)
```
</examples>

<patterns>
- Using deprecated io/ioutil (use io and os instead)
- Importing packages that are never referenced
- Using an import when the functionality exists in a smaller package
- Importing a package for a single function available elsewhere
</patterns>
