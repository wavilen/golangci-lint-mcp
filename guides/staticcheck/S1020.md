# staticcheck: S1020

<instructions>
Detects unnecessary `if` statements that wrap a boolean literal return. If the condition is already a boolean, return it directly — wrapping it in an if/else with true/false literals is redundant.

Simplify to a direct return of the boolean expression.
</instructions>

<examples>
## Bad
```go
func isValid() bool {
    if len(items) > 0 {
        return true
    } else {
        return false
    }
}
```

## Good
```go
func isValid() bool {
    return len(items) > 0
}
```
</examples>

<patterns>
- Return the boolean condition directly instead of if-else returning true/false
- Replace `if !x { return true } return false` with `return !x`
- Return the inverted boolean directly instead of using if to negate before returning
- Simplify overly verbose conditional boolean returns to direct returns
</patterns>
