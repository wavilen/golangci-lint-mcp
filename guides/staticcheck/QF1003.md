# staticcheck: QF1003

<instructions>
Detects uses of `strings.Replace` with the count argument set to `-1` (replace all). Since Go 1.12, `strings.ReplaceAll` is the clearer equivalent. The same applies to `bytes.Replace` → `bytes.ReplaceAll`.

Use `strings.ReplaceAll` or `bytes.ReplaceAll` instead.
</instructions>

<examples>
## Bad
```go
result := strings.Replace(s, "old", "new", -1)
```

## Good
```go
result := strings.ReplaceAll(s, "old", "new")
```
</examples>

<patterns>
- strings.Replace with -1 as count → use strings.ReplaceAll
- bytes.Replace with -1 as count → use bytes.ReplaceAll
- Using Replace when all occurrences should be replaced
- Missing the more descriptive ReplaceAll API
</patterns>
