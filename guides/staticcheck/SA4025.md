# staticcheck: SA4025

<instructions>
Detects integer division results assigned to float variables, such as `float64(1 / 2)` which yields `0.0` instead of `0.5`. The integer division is performed before the conversion, truncating the fractional part. Convert the operands to floats before dividing.

Convert at least one operand to a float type before the division: `float64(a) / float64(b)`.
</instructions>

<examples>
## Bad
```go
ratio := float64(total / count) // integer division, then conversion
```

## Good
```go
ratio := float64(total) / float64(count) // float division
```
</examples>

<patterns>
- `float64(a / b)` where both `a` and `b` are integers
- Percentage calculations: `float64(hits / total) * 100`
- Averaging integers: `float64(sum / n)`
- Converting after division instead of before
</patterns>
