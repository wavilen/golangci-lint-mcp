# staticcheck: SA1023

<instructions>
Detects an `io.Writer` implementation that modifies the input byte slice. The `io.Writer` contract specifies that the implementation must not modify the input data — it must not retain or mutate the slice. Modifying the input violates the `io.Writer` interface contract and causes subtle bugs in callers.
</instructions>

<examples>
## Bad
```go
func (w *myWriter) Write(p []byte) (int, error) {
    for i := range p {
        p[i] = strings.ToUpper(string(p[i]))[0] // modifies caller's data
    }
    return w.dst.Write(p)
}
```

## Good
```go
func (w *myWriter) Write(p []byte) (int, error) {
    upper := strings.ToUpper(string(p))
    return w.dst.Write([]byte(upper))
}
```
</examples>

<patterns>
- Mutating the `p []byte` parameter inside a `Write` method
- Using the input slice as a scratch buffer for transformations
- Storing references to the input slice past the `Write` call
</patterns>
