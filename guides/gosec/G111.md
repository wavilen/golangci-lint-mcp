# G111

<instructions>
Detects `http.Dir("/")` or `http.Dir("")` which serves the entire filesystem via HTTP. This exposes sensitive files like `/etc/passwd`, private keys, and environment files to any client.

Serve from a specific, restricted directory. Use `http.Dir("./static")` or a path derived from configuration, never root.
</instructions>

<examples>
## Bad
```go
fs := http.FileServer(http.Dir("/"))
http.Handle("/", fs)
```

## Good
```go
fs := http.FileServer(http.Dir("/var/www/static"))
http.Handle("/", fs)
```
</examples>

<patterns>
- Restrict `http.Dir` to a specific application directory — never use `"/"` as the root
- Set an explicit root path in `http.Dir` — avoid empty string which defaults to the working directory
- Limit file server scope to a dedicated static assets directory with no sensitive files
</patterns>
