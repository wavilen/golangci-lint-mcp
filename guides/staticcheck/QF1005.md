# staticcheck: QF1005

<instructions>
Detects code that builds a formatted string via `fmt.Sprintf` and then passes it to `log.Printf` or similar functions. This creates double formatting — use the format verb directly in the log function call.

Use `log.Printf` with the format string directly instead of wrapping with Sprintf.
</instructions>

<examples>
## Bad
```go
log.Printf(fmt.Sprintf("processing %d items", n))
```

## Good
```go
log.Printf("processing %d items", n)
```
</examples>

<patterns>
- fmt.Sprintf inside log.Printf — double formatting
- fmt.Sprintf inside fmt.Printf — use format verbs directly
- Building formatted strings then passing to functions that accept format
- Wrapping Sprintf output in functions that already support formatting
</patterns>
