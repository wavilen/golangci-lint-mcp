# staticcheck: SA4029

<instructions>
Detects struct field tags that are not recognized by any standard or popular package. This typically means a typo in the tag name (e.g., `"jason"` instead of `"json"`) or a tag that is never used by any known library.

Fix the tag name to match the intended package's convention, or remove the unused tag.
</instructions>

<examples>
## Good
```go
type User struct {
    Name string `json:"name"`
}
```
</examples>

<patterns>
- Fix typos in struct tags — `"jason"` → `"json"`, check `"bson"` spelling
- Remove tags for packages no longer used in the project
- Remove custom tags without corresponding processing code
- Fix copy-paste errors from other struct definitions
</patterns>
