# staticcheck: ST1007

<instructions>
Detects receiver names that are longer than necessary. Go convention uses short (1-2 character) receiver names. The receiver name should be a short abbreviation of the type name, not the full name.

Shorten the receiver name to 1-2 characters that abbreviate the type.
</instructions>

<examples>
## Bad
```go
func (server *Server) Start() {}
```

## Good
```go
func (s *Server) Start() {}
```
</examples>

<patterns>
- Using the full type name as receiver (server, client, handler)
- Receiver names longer than 2-3 characters
- Using "self" or "this" as receiver names (non-idiomatic)
- Receiver names that shadow imported package names
</patterns>
