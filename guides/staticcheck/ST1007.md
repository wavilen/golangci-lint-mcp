# staticcheck: ST1007

<instructions>
Detects receiver names that are longer than necessary. Go convention uses short (1-2 character) receiver names. The receiver name should be a short abbreviation of the type name, not the full name.

Shorten the receiver name to 1-2 characters that abbreviate the type.
</instructions>

<examples>
## Bad
```go
func (server *Server) Start() {}
```

## Good
```go
func (s *Server) Start() {}
```
</examples>

<patterns>
- Shorten receiver names to 1-2 characters
- Limit receiver names to 2-3 characters maximum
- Replace self or this receiver names with Go-style abbreviations
- Avoid receiver names that shadow imported package names
</patterns>
