# staticcheck: S1016

<instructions>
Detects redundant type conversions where the expression is already of the target type. This adds noise without changing behavior — the conversion is a no-op at runtime.

Remove the unnecessary type conversion.
</instructions>

<examples>
## Bad
```go
type MyInt int
var x MyInt = 5
y := MyInt(x) // x is already MyInt
```

## Good
```go
type MyInt int
var x MyInt = 5
y := x
```
</examples>

<patterns>
- Converting a value to its own type: T(x) where x is already T
- Redundant string([]byte(s)) round-trip
- Unnecessary int(uint(x)) conversions between same-size types
- Type conversion added "just to be safe" when already correct
</patterns>
