# staticcheck: QF1008

<instructions>
Detects unnecessary type conversions where the source and target types are identical or can be inferred. This includes `string(s)` where `s` is already a string, or `int(x)` where `x` is already an int.

Remove the unnecessary type conversion.
</instructions>

<examples>
## Good
```go
var s string = "hello"
x := s
```
</examples>

<patterns>
- Remove no-op type conversions where the value is already the target type
- Eliminate unnecessary `string([]byte(s))` round-trip conversions
- Remove explicit type conversions where type inference handles it
- Eliminate redundant casts in assignment contexts
</patterns>
