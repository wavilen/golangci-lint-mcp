# staticcheck: SA1030

<instructions>
Detects `fmt.Sprintf` with the `%t` format verb applied to a non-boolean value. `%t` is specifically for booleans. When used with a non-bool type, Go applies the default `%v` formatting instead, which produces confusing output. Use the correct format verb for the actual type.
</instructions>

<examples>
## Bad
```go
val := 42
s := fmt.Sprintf("%t", val) // prints "42", not a boolean representation
```

## Good
```go
val := 42
s := fmt.Sprintf("%d", val) // correct verb for int
```
</examples>

<patterns>
- Using `%t` with integer values expecting truthy/falsy output
- Copy-pasting format strings from logging of booleans to other types
- Assuming `%t` converts non-bools to `"true"`/`"false"` strings
</patterns>
