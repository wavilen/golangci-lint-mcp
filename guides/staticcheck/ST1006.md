# staticcheck: ST1006

<instructions>
Detects inconsistent receiver names across methods of the same type. All methods on a given type should use the same receiver name for consistency and readability. If one method uses `s *Server`, all methods on `Server` should use `s`.

Use a consistent short receiver name across all methods of the same type.
</instructions>

<examples>
## Bad
```go
func (s *Server) Start() {}
func (srv *Server) Stop() {}
func (server *Server) Status() {}
```

## Good
```go
func (s *Server) Start() {}
func (s *Server) Stop() {}
func (s *Server) Status() {}
```
</examples>

<patterns>
- Different receiver names for methods on the same type
- Mixing short and long receiver names (s vs srv vs server)
- Receiver names that change between pointer and value receivers
- Generic receiver names like "self" or "this" (non-idiomatic Go)
</patterns>
