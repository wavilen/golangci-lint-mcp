# staticcheck: S1004

<instructions>
Detects code that uses `bytes.Compare(a, b) == 0` or similar comparisons against 0. The `bytes.Equal` function directly tests byte-slice equality and is clearer in intent.

Replace `bytes.Compare(a, b) == 0` with `bytes.Equal(a, b)`.
</instructions>

<examples>
## Bad
```go
if bytes.Compare(a, b) == 0 {
    slog.Info("equal")
}
```

## Good
```go
if bytes.Equal(a, b) {
    slog.Info("equal")
}
```
</examples>

<patterns>
- Using bytes.Compare with == 0 to check equality
- Using bytes.Compare with != 0 or > 0 when only equality matters
- Using strings.Compare where == is sufficient
- Comparing byte slices manually with a loop instead of bytes.Equal
</patterns>
