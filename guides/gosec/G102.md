# G102

<instructions>
Detects when a network listener binds to all interfaces (`0.0.0.0` or `::`) instead of a specific address like `127.0.0.1`. Binding to all interfaces exposes the service to every network the host is connected to.

Bind to `127.0.0.1` (localhost) for local-only services, or explicitly choose the interface address when broader access is needed.
</instructions>

<examples>
## Bad
```go
ln, _ := net.Listen("tcp", "0.0.0.0:8080")
```

## Good
```go
ln, _ := net.Listen("tcp", "127.0.0.1:8080")
```
</examples>

<patterns>
- Bind to `127.0.0.1` or `localhost` instead of `0.0.0.0` unless external access is explicitly required
- Set an explicit bind address in `http.ListenAndServe` — use `127.0.0.1:port` for local-only services
- Configure development servers with explicit bind addresses before deploying to production
</patterns>
