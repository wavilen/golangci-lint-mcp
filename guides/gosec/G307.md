# G307

<instructions>
Detects files created with `os.Create` or `os.OpenFile` without specifying restrictive permissions. The default file mode is `0666` (modified by umask), which is too permissive for files that may contain sensitive data.

Use `os.OpenFile` with explicit restrictive permissions, or set a restrictive umask. For sensitive files like keys and configs, use `0600`.
</instructions>

<examples>
## Good
```go
f, err := os.OpenFile("session.db", os.O_RDWR|os.O_CREATE|os.O_EXCL, 0600)
if err != nil {
    return err
}
```
</examples>

<patterns>
- Use `os.OpenFile` with `0600` permissions instead of `os.Create` for sensitive files
- Set restrictive permissions (`0600`) in `os.OpenFile` for configuration files
- Always specify restrictive file modes (`0600`) when creating database, key store, or log files
</patterns>
