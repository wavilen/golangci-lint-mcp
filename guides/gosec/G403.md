# G403

<instructions>
Detects RSA keys shorter than 2048 bits. Short RSA keys (512, 1024 bits) can be factored by attackers, breaking encryption and signature security.

Generate RSA keys with at least 2048 bits (prefer 4096 for long-term security). Use `rsa.GenerateKey(rand.Reader, 2048)` or stronger.
</instructions>

<examples>
## Good
```go
key, err := rsa.GenerateKey(rand.Reader, 4096)
if err != nil {
    return err
}
```
</examples>

<patterns>
- Use `rsa.GenerateKey(rand, 2048)` minimum for all RSA key generation
- Remove hardcoded weak RSA keys — generate fresh keys with `rsa.GenerateKey(rand, 2048)` or stronger
- Validate key size configuration — enforce minimum 2048 bits for RSA keys
</patterns>
