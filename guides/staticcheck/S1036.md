# staticcheck: S1036

<instructions>
Detects unnecessary parentheses around expressions that don't need them. This includes redundant grouping in simple expressions, function calls, and assignments.

Remove the unnecessary parentheses to improve readability.
</instructions>

<examples>
## Bad
```go
x := (a + b)
```

## Good
```go
x := a + b
```
</examples>

<patterns>
- Remove unnecessary parentheses around simple binary expressions with clear precedence
- Remove extra parentheses wrapping function call arguments
- Remove parentheses around single-variable expressions
- Remove redundant grouping in composite literal values
</patterns>
