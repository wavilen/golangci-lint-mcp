# G111

<instructions>
Detects `http.Dir("/")` or `http.Dir("")` which serves the entire filesystem via HTTP. This exposes sensitive files like `/etc/passwd`, private keys, and environment files to any client.

Serve from a specific, restricted directory. Use `http.Dir("./static")` or a path derived from configuration, never root.
</instructions>

<examples>
## Bad
```go
fs := http.FileServer(http.Dir("/"))
http.Handle("/", fs)
```

## Good
```go
fs := http.FileServer(http.Dir("/var/www/static"))
http.Handle("/", fs)
```
</examples>

<patterns>
- `http.Dir("/")` exposing the entire filesystem
- `http.Dir("")` which defaults to the current working directory (often project root)
- File servers serving from overly broad directory paths
</patterns>
