# staticcheck: ST1017

<instructions>
Detects unnecessary explicit type conversions where the compiler can infer the type. This includes converting between numeric types in const declarations or in contexts where the target type is known.

Remove the explicit type conversion and let the compiler handle it.
</instructions>

<examples>
## Bad
```go
const x = int(42)
```

## Good
```go
const x = 42
```
</examples>

<patterns>
- Remove explicit type conversions in const declarations
- Remove unnecessary int, string, or float64 conversions
- Remove type conversions in assignments to already-typed variables
- Remove redundant conversions in function call arguments
</patterns>

<related>
S1016, ST1013, S1036
