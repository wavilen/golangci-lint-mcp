# G506

<instructions>
Detects imports of `golang.org/x/crypto/md4`. MD4 is a predecessor to MD5 and is even more broken — collision attacks against MD4 are trivial. It provides zero security for any purpose.

Use `crypto/sha256` for general-purpose hashing. For password storage, use `golang.org/x/crypto/bcrypt` or `argon2`. There is no legitimate use case for MD4 in modern software.
</instructions>

<examples>
## Bad
```go
import "golang.org/x/crypto/md4"

func hash(data []byte) []byte {
    h := md4.New()
    h.Write(data)
    return h.Sum(nil)
}
```

## Good
```go
import "crypto/sha256"

func hash(data []byte) []byte {
    h := sha256.Sum256(data)
    return h[:]
}
```
</examples>

<patterns>
- `golang.org/x/crypto/md4` imported for any hashing purpose
- MD4 used in legacy authentication protocols (NTLM)
- MD4 used for compatibility with obsolete file formats
- MD4 used where a quick hash is needed without security requirements
</patterns>
