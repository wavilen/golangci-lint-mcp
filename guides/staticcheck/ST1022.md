# staticcheck: ST1022

<instructions>
Detects unnecessary statements or expressions in function bodies. This includes dead code after return statements, unreachable code, or expressions that have no side effects.

Remove the dead or unreachable code.
</instructions>

<examples>
## Bad
```go
func done() error {
    return nil
    log.Println("never reached") // dead code
}
```

## Good
```go
func done() error {
    return nil
}
```
</examples>

<patterns>
- Code after return, panic, or os.Exit that is never reached
- Unreachable case branches in switch statements
- Dead code in conditional branches that are always true/false
- Unreachable defer statements
</patterns>

<related>
