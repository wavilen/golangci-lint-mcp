# staticcheck: SA1001

<instructions>
Detects calls to `template.Template.Execute` where the template is a plain string with no actions or delimiters. A template with no `{{...}}` directives is just a static string — use `io.WriteString` or `fmt.Fprint` instead for clarity and performance.
</instructions>

<examples>
## Bad
```go
t := template.Must(template.New("msg").Parse("hello world"))
t.Execute(os.Stdout, nil) // no template actions, unnecessary overhead
```

## Good
```go
fmt.Fprint(os.Stdout, "hello world")
```
</examples>

<patterns>
- Replace static template strings with `io.WriteString` or `fmt.Fprint` when no template actions exist
- Avoid `text/template` or `html/template` for static output
- Replace over-engineered template usage for simple string output
</patterns>
