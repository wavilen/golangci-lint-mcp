# staticcheck: S1007

<instructions>
Detects manual URL path construction using string concatenation, `path.Join`, or `filepath.Join` with URL components. The `url.JoinPath` function handles trailing slashes and path segments correctly for URLs.

Replace manual URL path building with `url.JoinPath(base, parts...)`.
</instructions>

<examples>
## Good
```go
endpoint, err := url.JoinPath("https://example.com/api", "users", strconv.Itoa(id))
```
</examples>

<patterns>
- Replace URL concatenation with `+` using `url.JoinPath(base, parts...)`
- Use `url.JoinPath` instead of `path.Join` for URLs
- Replace manual API URL construction with `url.JoinPath`
- Avoid manual slash management between URL segments
</patterns>
