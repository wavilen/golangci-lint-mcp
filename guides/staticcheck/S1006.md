# staticcheck: S1006

<instructions>
Detects uses of `fmt.Printf` or `fmt.Sprintf` followed by `fmt.Println` or `fmt.Print` with a string that already contains formatting. More commonly, it flags `fmt.Print(fmt.Sprintf(...))` which should just be `fmt.Printf`.

Use `fmt.Printf` directly instead of combining format-then-print calls.
</instructions>

<examples>
## Good
```go
fmt.Printf("hello %s\n", name)
```
</examples>

<patterns>
- Replace `fmt.Println(fmt.Sprintf(...))` with `fmt.Printf`
- Replace `fmt.Print` with a pre-formatted string using `fmt.Printf`
- Use `fmt.Printf` directly instead of formatting then printing
- Combine format verbs directly into `Printf` instead of separate Print calls
</patterns>
