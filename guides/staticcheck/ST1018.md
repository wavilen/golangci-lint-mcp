# staticcheck: ST1018

<instructions>
Detects unnecessary code in control flow structures. This includes redundant else branches, unnecessary nesting, or patterns that could be simplified with early returns or guard clauses.

Flatten the control flow by removing unnecessary nesting.
</instructions>

<examples>
## Good
```go
func process(items []int) {
    for _, item := range items {
        handle(item)
    }
}
```
</examples>

<patterns>
- Flatten unnecessary nesting with early returns or guard clauses
- Simplify overly complex guard clauses
- Replace nested if chains with switch statements
- Remove redundant conditional nesting
</patterns>
