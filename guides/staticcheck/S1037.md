# staticcheck: S1037

<instructions>
Detects unnecessary else branches where the if body already handles the condition with a control flow statement. Similar to S1035 but specifically targets the structural redundancy of having an else at all.

Remove the else and de-indent the else block's code.
</instructions>

<examples>
## Bad
```go
if done {
    return result
} else {
    process()
}
```

## Good
```go
if done {
    return result
}
process()
```
</examples>

<patterns>
- Else block after a return in the if body
- Else block after a continue in a loop
- Else block after a break in a loop
- Else block after panic in the if body
</patterns>

<related>
S1035, SA5012, S1027
