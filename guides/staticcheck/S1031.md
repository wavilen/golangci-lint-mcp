# staticcheck: S1031

<instructions>
Detects unnecessary wrapping of values with `fmt.Sprintf` when the value can be used directly or with a simpler conversion. This is a broader check than S1029/S1030, catching cases where the formatting layer adds no value.

Remove the `fmt.Sprintf` wrapper and use the value directly.
</instructions>

<examples>
## Bad
```go
fmt.Println(fmt.Sprintf("value: %d", x))
```

## Good
```go
fmt.Printf("value: %d\n", x)
```
</examples>

<patterns>
- Replace `fmt.Println(fmt.Sprintf(...))` with `fmt.Printf`
- Pass format strings directly instead of pre-formatting with `Sprintf`
- Eliminate double formatting from `Sprintf` inside `Printf`
- Remove unnecessary `Sprintf` when direct string conversion works
</patterns>
