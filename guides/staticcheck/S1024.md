# staticcheck: S1024

<instructions>
Detects code that computes time until a deadline as `deadline.Sub(time.Now())` instead of using the idiomatic `time.Until(deadline)`. The `time.Until` helper is the counterpart to `time.Since` and is clearer in intent.

Replace `deadline.Sub(time.Now())` with `time.Until(deadline)`.
</instructions>

<examples>
## Bad
```go
remaining := deadline.Sub(time.Now())
```

## Good
```go
remaining := time.Until(deadline)
```
</examples>

<patterns>
- Using t.Sub(time.Now()) to compute time remaining
- Computing duration until a future time manually
- Using time.Now() and Sub where Until is the direct equivalent
- Measuring timeout durations with Sub instead of Until
</patterns>
