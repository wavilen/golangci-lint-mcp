# staticcheck: QF1010

<instructions>
Detects code that reimplements built-in functionality. This includes manually implementing `min`, `max`, `clear`, or other operations that Go's standard library provides as built-ins (Go 1.21+) or standard functions.

Use the built-in function or standard library equivalent.
</instructions>

<examples>
## Bad
```go
if a < b {
    min = a
} else {
    min = b
}
```

## Good
```go
min := min(a, b) // built-in since Go 1.21
```
</examples>

<patterns>
- Manual min/max implementations instead of built-in (Go 1.21+)
- Manual slice clearing instead of clear() built-in
- Reimplementing sort.Ints/sort.Strings manually
- Custom implementations of standard library functions
</patterns>
