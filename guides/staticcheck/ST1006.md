# staticcheck: ST1006

<instructions>
Detects inconsistent receiver names across methods of the same type. All methods on a given type should use the same receiver name for consistency and readability. If one method uses `s *Server`, all methods on `Server` should use `s`.

Use a consistent short receiver name across all methods of the same type.
</instructions>

<examples>
## Bad
```go
func (s *Server) Start() {}
func (srv *Server) Stop() {}
func (server *Server) Status() {}
```

## Good
```go
func (s *Server) Start() {}
func (s *Server) Stop() {}
func (s *Server) Status() {}
```
</examples>

<patterns>
- Use a consistent receiver name across all methods on the same type
- Standardize receiver name length
- Keep receiver names consistent between pointer and value receivers
- Replace non-idiomatic receiver names like self or this with Go-style abbreviations
</patterns>
