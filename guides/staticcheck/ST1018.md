# staticcheck: ST1018

<instructions>
Detects unnecessary code in control flow structures. This includes redundant else branches, unnecessary nesting, or patterns that could be simplified with early returns or guard clauses.

Flatten the control flow by removing unnecessary nesting.
</instructions>

<examples>
## Bad
```go
func process(items []int) {
    if len(items) > 0 {
        for _, item := range items {
            handle(item)
        }
    }
}
```

## Good
```go
func process(items []int) {
    for _, item := range items {
        handle(item)
    }
}
```
</examples>

<patterns>
- Unnecessary nesting that could be flattened with early return
- Guard clauses that are overly complex
- Control flow that could use switch instead of nested if
- Redundant conditional nesting
</patterns>
