# staticcheck: S1026

<instructions>
Detects redundant type assertions where the value is already known to be of the asserted type. This can happen after a type switch or when the assertion result is immediately used as the same type.

Remove the redundant type assertion and use the value directly.
</instructions>

<examples>
## Good
```go
var s string = "hello"
x := s
```
</examples>

<patterns>
- Remove type assertions on values that are already the asserted type
- Remove double type assertions after extracting in a type switch
- Remove `x.(T)` assertions when code already guarantees `x` is type `T`
- Eliminate redundant assertions on narrow interface types
</patterns>
