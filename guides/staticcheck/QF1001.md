# staticcheck: QF1001

<instructions>
Detects boolean expressions that can be simplified by applying De Morgan's law. Expressions like `!a && !b` should become `!(a || b)`, and `!a || !b` should become `!(a && b)`. This makes the logic clearer and more compact.

Apply De Morgan's law to simplify the compound boolean expression.
</instructions>

<examples>
## Bad
```go
if !a && !b {
    return true
}
```

## Good
```go
if !(a || b) {
    return true
}
```
</examples>

<patterns>
- Apply De Morgan's law: replace `!a && !b` with `!(a || b)`
- Apply De Morgan's law: replace `!a || !b` with `!(a && b)`
- Eliminate double negations that can be collapsed
- Simplify compound boolean expressions with repeated negation
</patterns>
