# staticcheck: SA5009

<instructions>
Detects `fmt.Sprintf` calls where the result is only used as a format string for another print function, such as `fmt.Printf(fmt.Sprintf(...))`. This is redundant — the formatting can be done directly in the outer call.

Combine the formatting into a single call by using the outer function's format string directly.
</instructions>

<examples>
## Good
```go
fmt.Printf("Hello %s", name)
```
</examples>

<patterns>
- Replace `fmt.Printf(fmt.Sprintf(...))` with format verbs directly in `Printf`
- Replace `fmt.Println(fmt.Sprintf(...))` with `Printf` or `Println` directly
- Replace `log.Printf(fmt.Sprintf(...))` with format verbs in `log.Printf` directly
- Remove dynamic format string building when a static format string suffices
</patterns>
