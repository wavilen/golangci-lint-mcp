# staticcheck: ST1016

<instructions>
Detects uses of `interface{}` where `any` (introduced in Go 1.18) is preferred. The `any` type alias is more concise and is the modern Go convention for denoting arbitrary types.

Replace `interface{}` with `any`.
</instructions>

<examples>
## Bad
```go
func process(v interface{}) {}
```

## Good
```go
func process(v any) {}
```
</examples>

<patterns>
- Using interface{} instead of the any alias (Go 1.18+)
- Map[string]interface{} instead of map[string]any
- []interface{} instead of []any
- Type assertions on interface{} values
</patterns>
