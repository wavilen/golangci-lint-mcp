# G306

<instructions>
Detects writing to files with overly permissive permissions. Using `0666` or similar when writing files allows any local user to read or modify the file contents.

Specify restrictive permissions when writing files. Use `0600` for private data (keys, tokens) and `0640` for shared-but-not-public files.
</instructions>

<examples>
## Bad
```go
os.WriteFile("output.json", data, 0666)
```

## Good
```go
os.WriteFile("output.json", data, 0640)
```
</examples>

<patterns>
- Set file permissions to `0600` or `0644` in `os.WriteFile` — never use `0666`
- Restrict `os.OpenFile` mode to `0600` for sensitive files — avoid permissive `0666` modes
- Configure explicit file permissions when creating log and data files — do not rely on umask defaults
</patterns>
