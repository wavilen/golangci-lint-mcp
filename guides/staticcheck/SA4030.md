# staticcheck: SA4030

<instructions>
Detects use of the deprecated `strings.Title` function. `strings.Title` does not handle Unicode properly and does not follow title-casing rules for all languages. It has been deprecated since Go 1.18.

Replace with `cases.Title` from the `golang.org/x/text/cases` package, which handles Unicode correctly.
</instructions>

<examples>
## Bad
```go
title := strings.Title("hello world")
```

## Good
```go
import "golang.org/x/text/cases"
import "golang.org/x/text/language"

title := cases.Title(language.English).String("hello world")
```
</examples>

<patterns>
- Using `strings.Title` for title-casing user-facing text
- Not importing `golang.org/x/text/cases` as a replacement
- Code written before Go 1.18 that hasn't been updated
- Assuming `strings.Title` handles non-ASCII correctly
</patterns>
