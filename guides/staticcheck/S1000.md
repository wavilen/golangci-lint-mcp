# staticcheck: S1000

<instructions>
Detects `select` statements with a single case that sends a value on a channel. Using `select` for a single channel send is unnecessary — a plain send statement is simpler and more readable.

Replace the `select` block with a direct channel send.
</instructions>

<examples>
## Good
```go
ch := make(chan int)
ch <- 42
```
</examples>

<patterns>
- Replace single-case `select` with a direct channel send
- Remove `select` with only a default case wrapping a send
- Simplify copy-pasted `select` that originally had multiple cases
- Avoid using `select` for documentation purposes when a plain send suffices
</patterns>
