# staticcheck: S1016

<instructions>
Detects redundant type conversions where the expression is already of the target type. This adds noise without changing behavior — the conversion is a no-op at runtime.

Remove the unnecessary type conversion.
</instructions>

<examples>
## Good
```go
type MyInt int
var x MyInt = 5
y := x
```
</examples>

<patterns>
- Remove no-op type conversions `T(x)` where `x` is already type `T`
- Eliminate redundant `string([]byte(s))` round-trip conversions
- Remove unnecessary `int(uint(x))` conversions between same-size types
- Remove type conversions added just to be safe when already correct
</patterns>
