# staticcheck: S1039

<instructions>
Detects unnecessary statements or expressions that have no effect. This includes unused result values from pure functions, expressions computed but never stored, and other dead code patterns.

Remove or restructure the unnecessary statement.
</instructions>

<examples>
## Bad
```go
x + 1 // computed but result discarded
```

## Good
```go
x = x + 1
// or: y := x + 1
```
</examples>

<patterns>
- Expressions computed but results never stored
- Pure function calls whose return values are ignored
- Arithmetic operations without assignment
- Unused intermediate calculations
</patterns>

<related>
S1018, S1005, SA4010
