# G406

<instructions>
Detects use of deprecated hash algorithms MD4 (`golang.org/x/crypto/md4`) or RIPEMD-160 (`golang.org/x/crypto/ripemd160`). MD4 is fully broken with trivial collisions. RIPEMD-160 is deprecated for new applications and has known weaknesses.

Use SHA-256 or SHA-3 from the standard library for new implementations. Only retain MD4/RIPEMD-160 for legacy protocol compatibility behind strict feature flags.
</instructions>

<examples>
## Good
```go
import "crypto/sha256"

hash := sha256.Sum256(data)
```
</examples>

<patterns>
- Remove `golang.org/x/crypto/md4` — use `crypto/sha256` for any hashing purpose
- Replace `golang.org/x/crypto/ripemd160` with `crypto/sha256` unless protocol-mandated
- Replace legacy protocol hashes with SHA-256 or SHA-3 where possible
</patterns>
