# staticcheck: SA9005

<instructions>
Detects attempts to marshal or encode unexported (lowercase) struct fields using `encoding/json`, `encoding/xml`, or similar packages. Unexported fields are invisible to the reflect-based encoders and will be silently omitted from the output.

Export the field (capitalize the first letter), or add an explicit unexported-field workaround if the field must remain private.
</instructions>

<examples>
## Good
```go
type User struct {
    Name string `json:"name"` // exported, appears in JSON
    Age  int    `json:"age"`
}
```
</examples>

<patterns>
- Export struct fields (capitalize first letter) that have json/xml tags but are unexported
- Export private fields expected to appear in serialized output
- Fix `encoding/json.Marshal` incomplete output by exporting tagged fields
- Remove json/xml tags from unexported fields — tags do not override Go visibility rules
</patterns>
