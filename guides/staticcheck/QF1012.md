# staticcheck: QF1012

<instructions>
Detects code patterns that can be simplified by removing unnecessary intermediate steps. This includes redundant variable assignments, unnecessary temporary values, or code that could be written more directly.

Inline or remove the unnecessary intermediate steps.
</instructions>

<examples>
## Bad
```go
tmp := getValue()
result := tmp
```

## Good
```go
result := getValue()
```
</examples>

<patterns>
- Assigning to a temp variable then immediately reassigning
- Unnecessary intermediate variable for a simple expression
- Redundant copy before return
- Two-step initialization that can be one step
</patterns>
