# staticcheck: S1039

<instructions>
Detects unnecessary statements or expressions that have no effect. This includes unused result values from pure functions, expressions computed but never stored, and other dead code patterns.

Remove or restructure the unnecessary statement.
</instructions>

<examples>
## Bad
```go
x + 1 // computed but result discarded
```

## Good
```go
x = x + 1
// or: y := x + 1
```
</examples>

<patterns>
- Store or remove expressions whose results are computed but never used
- Handle or remove pure function calls whose return values are ignored
- Assign or remove arithmetic operations without assignment
- Use or remove unused intermediate calculations
</patterns>

<related>
S1018, S1005, SA4010
