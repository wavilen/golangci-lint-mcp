# staticcheck: S1003

<instructions>
Detects a chain of `if`/`else if` statements that compare the same variable against different values. A `switch` statement is the idiomatic Go construct for this pattern — it is more readable and avoids repeated variable references.

Replace the if/else chain with a `switch` on the variable.
</instructions>

<examples>
## Bad
```go
if x == 1 {
    slog.Info("one")
} else if x == 2 {
    slog.Info("two")
} else if x == 3 {
    slog.Info("three")
}
```

## Good
```go
switch x {
case 1:
    slog.Info("one")
case 2:
    slog.Info("two")
case 3:
    slog.Info("three")
}
```
</examples>

<patterns>
- Replace multiple `if`/`else if` branches with a `switch` on the same variable
- Convert chained `if` statements comparing a string to several values into `switch`
- Replace equality comparisons that could be switch cases with `switch`
- Use `switch` for `if`/`else chains with 3 or more branches on the same expression
</patterns>
