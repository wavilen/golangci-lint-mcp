# staticcheck: QF1005

<instructions>
Detects code that builds a formatted string via `fmt.Sprintf` and then passes it to `log.Printf` or similar functions. This creates double formatting — use the format verb directly in the log function call.

Use `log.Printf` with the format string directly instead of wrapping with Sprintf.
</instructions>

<examples>
## Bad
```go
log.Printf(fmt.Sprintf("processing %d items", n))
```

## Good
```go
log.Printf("processing %d items", n)
```
</examples>

<patterns>
- Replace `log.Printf(fmt.Sprintf(...))` with direct format verbs in `log.Printf`
- Replace `fmt.Printf(fmt.Sprintf(...))` with format verbs directly in Printf
- Pass format verbs directly instead of pre-formatting with `Sprintf`
- Remove `Sprintf` wrapper when the outer function already supports formatting
</patterns>
