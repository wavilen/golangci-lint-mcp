# staticcheck: S1021

<instructions>
Detects code that declares a variable with `var` and then immediately assigns a composite literal. The declaration and initialization can be combined into a single `:=` statement with the composite literal.

Use short variable declaration with the composite literal directly.
</instructions>

<examples>
## Bad
```go
var m map[string]int
m = map[string]int{}
```

## Good
```go
m := map[string]int{}
```
</examples>

<patterns>
- Combine `var` declaration and composite literal assignment into `:=`
- Replace separate `var` + assignment with a short declaration
- Merge separate declaration and initialization of slices, maps, or structs
- Use `:=` with composite literals instead of `var` when the type is inferrable
</patterns>
