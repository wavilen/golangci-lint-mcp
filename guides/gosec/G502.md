# G502

<instructions>
Detects imports of `crypto/des`. DES uses a 56-bit key that is trivially brute-forced with modern hardware. It provides no meaningful security for encrypted data.

Use AES (`crypto/aes` with `crypto/cipher` for GCM mode) for symmetric encryption. AES-256-GCM provides both encryption and authentication in a single operation.
</instructions>

<examples>
## Good
```go
import (
    "crypto/aes"
    "crypto/cipher"
    "crypto/rand"
)

func encrypt(key []byte, plaintext []byte) ([]byte, error) {
    block, err := aes.NewCipher(key)
    if err != nil {
        return nil, fmt.Errorf("creating cipher: %w", err)
    }
    gcm, err := cipher.NewGCM(block)
    if err != nil {
        return nil, fmt.Errorf("creating GCM: %w", err)
    }
    nonce := make([]byte, gcm.NonceSize())
    _, readErr := rand.Read(nonce)
    if readErr != nil {
        return nil, fmt.Errorf("generating nonce: %w", readErr)
    }
    return gcm.Seal(nonce, nonce, plaintext, nil), nil
}
```
</examples>

<patterns>
- Remove `crypto/des` imports — use `crypto/aes` for all encryption needs
- Replace Triple DES with AES-GCM — 3DES is deprecated and insecure
- Replace legacy DES protocols with AES — wrap legacy systems with AES-based encryption at the boundary
- Replace DES/ECB with AES-GCM immediately — ECB mode reveals data patterns and DES is brute-forceable
</patterns>
