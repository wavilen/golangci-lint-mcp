# staticcheck: QF1010

<instructions>
Detects code that reimplements built-in functionality. This includes manually implementing `min`, `max`, `clear`, or other operations that Go's standard library provides as built-ins (Go 1.21+) or standard functions.

Use the built-in function or standard library equivalent.
</instructions>

<examples>
## Good
```go
min := min(a, b) // built-in since Go 1.21
```
</examples>

<patterns>
- Replace manual min/max implementations with built-in `min`/`max` (Go 1.21+)
- Replace manual slice clearing with the built-in `clear()` function
- Use `sort.Ints`/`sort.Strings` instead of manual sort implementations
- Replace custom stdlib reimplementations with the standard library versions
</patterns>
