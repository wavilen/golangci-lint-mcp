# staticcheck: SA4029

<instructions>
Detects struct field tags that are not recognized by any standard or popular package. This typically means a typo in the tag name (e.g., `"jason"` instead of `"json"`) or a tag that is never used by any known library.

Fix the tag name to match the intended package's convention, or remove the unused tag.
</instructions>

<examples>
## Bad
```go
type User struct {
    Name string `jason:"name"` // typo: "jason" instead of "json"
}
```

## Good
```go
type User struct {
    Name string `json:"name"`
}
```
</examples>

<patterns>
- Typos in struct tags: `"jason"`, `"bson"` misspelled
- Tags for packages no longer used in the project
- Custom tags without corresponding processing code
- Copy-paste errors from other struct definitions
</patterns>
