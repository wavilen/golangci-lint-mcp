# staticcheck: S1008

<instructions>
Detects functions that return a boolean using an if/else that returns `true` or `false` literally. The condition itself can be returned directly — the comparison expression already evaluates to a boolean.

Replace the if/return true/return false pattern with a single return statement.
</instructions>

<examples>
## Bad
```go
func isEven(n int) bool {
    if n%2 == 0 {
        return true
    }
    return false
}
```

## Good
```go
func isEven(n int) bool {
    return n%2 == 0
}
```
</examples>

<patterns>
- Returning true/false based on a boolean condition
- If-else returning opposite boolean literals
- Negating a condition with if/else instead of using !
- Functions where the entire body is a conditional return of boolean literals
</patterns>
