# staticcheck: SA1003

<instructions>
Detects calls to `binary.Read` or `binary.Write` with a value that cannot be decoded or encoded. Passing a non-pointer to `binary.Read`, or a type without fixed size (maps, slices, non-empty interfaces) causes a runtime error.

Ensure the destination argument to `binary.Read` is a pointer to a fixed-size type or a struct composed of fixed-size fields.
</instructions>

<examples>
## Bad
```go
var buf bytes.Buffer
var val uint32
binary.Read(&buf, binary.LittleEndian, val) // val is not a pointer
```

## Good
```go
var buf bytes.Buffer
var val uint32
binary.Read(&buf, binary.LittleEndian, &val) // pointer to fixed-size type
```
</examples>

<patterns>
- Pass a pointer (not a value) to `binary.Read` as the destination argument
- Avoid `binary.Read` with slices, maps, or interface types
- Ensure structs passed to `binary.Read` contain only fixed-size fields
</patterns>
