# staticcheck: S1008

<instructions>
Detects functions that return a boolean using an if/else that returns `true` or `false` literally. The condition itself can be returned directly — the comparison expression already evaluates to a boolean.

Replace the if/return true/return false pattern with a single return statement.
</instructions>

<examples>
## Good
```go
func isEven(n int) bool {
    return n%2 == 0
}
```
</examples>

<patterns>
- Return the boolean condition directly instead of `if { return true } return false`
- Replace if-else returning opposite boolean literals with a direct return
- Use `!` negation instead of if/else to invert a boolean before returning
- Simplify functions whose entire body is a conditional return of boolean literals
</patterns>
