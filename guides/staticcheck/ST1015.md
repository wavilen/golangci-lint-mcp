# staticcheck: ST1015

<instructions>
Detects unnecessary code patterns in function and method declarations. This includes redundant parameter names, unnecessary grouping, or patterns that obscure the code's intent.

Simplify the declaration by removing unnecessary patterns.
</instructions>

<examples>
## Good
```go
func (s *Server) handle(req *Request) {}
```
</examples>

<patterns>
- Remove duplicate parameter names that shadow the receiver in method declarations
- Remove redundant parameter declarations in function signatures
- Avoid unnecessary parameter grouping
- Follow idiomatic Go function signature style
</patterns>
