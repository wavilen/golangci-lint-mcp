# staticcheck: ST1012

<instructions>
Detects poor style patterns that don't fit other categories. This includes writing `context.Background()` as the first argument to a function when the context should be passed through, or using poor signal-to-noise patterns in code.

Follow idiomatic Go patterns and conventions.
</instructions>

<examples>
## Good
```go
func handler(w http.ResponseWriter, r *http.Request) {
    process(r.Context(), r)
}
```
</examples>

<patterns>
- Use available request context instead of creating `context.Background()`
- Replace `context.TODO()` in production code with properly propagated contexts
- Follow idiomatic Go patterns for code clarity
- Replace non-idiomatic Go patterns inherited from other languages
</patterns>
