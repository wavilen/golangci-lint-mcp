# staticcheck: QF1009

<instructions>
Detects unnecessary code that can be removed without affecting behavior. This includes redundant nil checks, unnecessary assignments, or dead code that the compiler can optimize away but clutters the source.

Remove the unnecessary code entirely.
</instructions>

<examples>
## Bad
```go
if x == x {
    doSomething()
}
```

## Good
```go
doSomething()
```
</examples>

<patterns>
- Remove self-comparisons (`x == x` is always true, `x != x` is always false)
- Remove unnecessary boolean conversions in conditionals
- Eliminate dead code after unconditional returns
- Remove redundant nil checks on guaranteed-non-nil values
</patterns>
