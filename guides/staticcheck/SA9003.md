# staticcheck: SA9003

<instructions>
Detects empty conditional branches such as `if cond {}` with no body, or `if cond { } else { }` where both branches are empty. This is dead code that serves no purpose and is likely the result of incomplete implementation.

Remove the empty branch, implement the missing logic, or invert the condition to simplify the code.
</instructions>

<examples>
## Bad
```go
if err != nil {
    // TODO: handle error
}
```

## Good
```go
if err != nil {
    return errors.Wrap(err, "operation failed")
}
```
</examples>

<patterns>
- `if` statements with only comments inside the body
- Empty `if/else` blocks from incomplete refactoring
- TODO placeholder branches not yet implemented
- Conditionals that were needed but logic was removed
</patterns>

<related>
SA5012, SA4026
</related>
