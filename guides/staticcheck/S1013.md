# staticcheck: S1013

<instructions>
Detects code that manually computes seek offsets using arithmetic instead of using the `io.SeekStart`, `io.SeekCurrent`, and `io.SeekEnd` constants. These constants were introduced in Go 1.7 to replace magic numbers.

Use the named constants from the `io` package for seek whence values.
</instructions>

<examples>
## Bad
```go
file.Seek(0, 2) // seek to end
```

## Good
```go
file.Seek(0, io.SeekEnd)
```
</examples>

<patterns>
- Replace magic numbers `0`, `1`, `2` in `Seek` with `io.SeekStart`, `io.SeekCurrent`, `io.SeekEnd`
- Use `io.Seek*` constants instead of manually computing seek positions
- Replace deprecated `os.SEEK_*` aliases with `io.Seek*` constants
- Replace hardcoded offset constants in `Seek` calls with named constants
</patterns>
