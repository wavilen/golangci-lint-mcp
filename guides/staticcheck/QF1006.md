# staticcheck: QF1006

<instructions>
Detects compound expressions that can be simplified. This includes arithmetic simplifications, unnecessary intermediate values, and expressions where the computation can be reduced to a simpler form.

Simplify the expression to its most reduced form.
</instructions>

<examples>
## Bad
```go
x := a * 1 + b * 0
```

## Good
```go
x := a
```
</examples>

<patterns>
- Arithmetic identity: x * 1, x + 0, x / 1 — all no-ops
- Dead terms: x * 0, x - x — always zero
- Redundant operations that cancel out
- Complex expressions with trivial simplifications
</patterns>
