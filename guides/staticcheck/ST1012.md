# staticcheck: ST1012

<instructions>
Detects poor style patterns that don't fit other categories. This includes writing `context.Background()` as the first argument to a function when the context should be passed through, or using poor signal-to-noise patterns in code.

Follow idiomatic Go patterns and conventions.
</instructions>

<examples>
## Bad
```go
func handler(w http.ResponseWriter, r *http.Request) {
    ctx := context.Background()
    process(ctx, r)
}
```

## Good
```go
func handler(w http.ResponseWriter, r *http.Request) {
    process(r.Context(), r)
}
```
</examples>

<patterns>
- Creating context.Background() when request context is available
- Using context.TODO() in production code
- Poor style patterns that reduce code clarity
- Non-idiomatic Go patterns inherited from other languages
</patterns>
