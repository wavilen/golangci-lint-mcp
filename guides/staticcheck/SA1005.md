# staticcheck: SA1005

<instructions>
Detects invalid time format strings passed to `time.Format`. The argument must use Go's reference time `Mon Jan 2 15:04:05 MST 2006`. If the format string contains components that don't match the reference time (e.g., `YYYY` or `DD`), the output will be wrong silently.

Replace non-standard format tokens with Go reference time components.
</instructions>

<examples>
## Good
```go
formatted := time.Now().Format("2006-01-02")
```
</examples>

<patterns>
- Replace `YYYY` with `2006` for year in `time.Format` strings
- Replace `MM` or `DD` with `01` and `02` in Go time format strings
- Replace strftime conventions in Go `Format` calls with Go reference time tokens
- Replace format strings copy-pasted from other languages with Go equivalents
</patterns>
