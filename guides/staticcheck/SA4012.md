# staticcheck: SA4012

<instructions>
Detects the `!` (not) operator applied to a channel value. Channels are never boolean — applying `!` to a channel is a type error in most cases or indicates confusion with `select`. This code will not compile or does not do what the author intended.

Use a `select` statement to check channel readiness, or a boolean variable instead of a channel for the condition.
</instructions>

<examples>
## Good
```go
ch := make(chan int)
select {
case <-ch:
    slog.Info("received")
default:
    slog.Info("no data")
}
```
</examples>

<patterns>
- Replace `!ch` with a `select` statement when `ch` is a channel
- Replace channel-as-boolean patterns with a proper done/cancel boolean flag
- Use `select` to check channel readiness instead of boolean negation
- Distinguish `context.Done()` channel from a boolean cancel flag
</patterns>
