# staticcheck: S1004

<instructions>
Detects code that uses `bytes.Compare(a, b) == 0` or similar comparisons against 0. The `bytes.Equal` function directly tests byte-slice equality and is clearer in intent.

Replace `bytes.Compare(a, b) == 0` with `bytes.Equal(a, b)`.
</instructions>

<examples>
## Bad
```go
if bytes.Compare(a, b) == 0 {
    slog.Info("equal")
}
```

## Good
```go
if bytes.Equal(a, b) {
    slog.Info("equal")
}
```
</examples>

<patterns>
- Replace `bytes.Compare(a, b) == 0` with `bytes.Equal(a, b)`
- Replace `bytes.Compare` with `!= 0` or `> 0` when only equality matters
- Replace `strings.Compare` with `==` when direct comparison suffices
- Replace manual byte-slice comparison loops with `bytes.Equal`
</patterns>
