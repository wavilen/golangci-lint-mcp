# staticcheck: S1036

<instructions>
Detects unnecessary parentheses around expressions that don't need them. This includes redundant grouping in simple expressions, function calls, and assignments.

Remove the unnecessary parentheses to improve readability.
</instructions>

<examples>
## Bad
```go
x := (a + b)
```

## Good
```go
x := a + b
```
</examples>

<patterns>
- Parentheses around simple binary expressions where precedence is unambiguous
- Wrapping function call arguments in extra parentheses
- Parentheses around single-variable expressions
- Redundant grouping in composite literal values
</patterns>

<related>
