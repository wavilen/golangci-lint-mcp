# staticcheck: ST1014

<instructions>
Detects unnecessary import aliases or import groupings that could be simplified. This includes importing a package under its own name, or splitting import groups unnecessarily.

Use the default package name or consolidate import groups.
</instructions>

<examples>
## Bad
```go
import (
    fmt "fmt"
    "strings"
)
```

## Good
```go
import (
    "fmt"
    "strings"
)
```
</examples>

<patterns>
- Importing a package aliased to its own name
- Unnecessary import aliases that add no clarity
- Splitting import groups without reason
- Importing packages that are only used for side effects without blank import
</patterns>

<related>
ST1001, S1036, SA1019
