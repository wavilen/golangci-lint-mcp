# staticcheck: SA6006

<instructions>
Detects uses of `fmt.Fprintf`, `fmt.Printf`, or similar where `io.WriteString` would be simpler and more efficient. When writing a plain string (no format verbs) to an `io.Writer`, `io.WriteString` avoids the overhead of format parsing and is significantly faster.

Use `io.WriteString(w, s)` instead of `fmt.Fprintf(w, "%s", s)` or `fmt.Fprint(w, s)` when no formatting is needed.
</instructions>

<examples>
## Bad
```go
fmt.Fprintf(w, "Hello, World!")
```

## Good
```go
io.WriteString(w, "Hello, World!")
```
</examples>

<patterns>
- `fmt.Fprintf(w, s)` where `s` has no format verbs
- `fmt.Fprint(w, stringValue)` for writing plain strings
- `fmt.Fprintf(w, "%s", s)` instead of `io.WriteString(w, s)`
- Using `fmt.Fprintf` for static string output in HTTP handlers
</patterns>
