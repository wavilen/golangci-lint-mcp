# staticcheck: S1018

<instructions>
Detects self-assignment where a variable is assigned to itself, such as `x = x`. This is a no-op and likely indicates a typo or incomplete refactoring.

Remove the self-assignment or fix it to assign the intended value.
</instructions>

<examples>
## Bad
```go
name := "Alice"
name = name // no-op
```

## Good
```go
name := "Alice"
name = "Bob"
```
</examples>

<patterns>
- Remove self-assignments `x = x` — fix or delete the no-op
- Remove self-assignments in struct methods like `s.field = s.field`
- Fix copy-paste errors where the right-hand side was meant to be different
- Remove leftover debugging or partial refactoring self-assignments
</patterns>
