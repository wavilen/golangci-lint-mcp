# staticcheck: S1020

<instructions>
Detects unnecessary `if` statements that wrap a boolean literal return. If the condition is already a boolean, return it directly — wrapping it in an if/else with true/false literals is redundant.

Simplify to a direct return of the boolean expression.
</instructions>

<examples>
## Bad
```go
func isValid() bool {
    if len(items) > 0 {
        return true
    } else {
        return false
    }
}
```

## Good
```go
func isValid() bool {
    return len(items) > 0
}
```
</examples>

<patterns>
- If-else returning true and false based on a boolean condition
- Wrapping a negation: if !x { return true } return false
- Using if to invert a boolean before returning
- Overly verbose conditional boolean returns
</patterns>
