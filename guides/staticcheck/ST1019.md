# staticcheck: ST1019

<instructions>
Detects unnecessary imports that are either unused or could be replaced by a more specific import. This includes importing packages that provide overlapping functionality.

Remove unused imports or use more appropriate packages.
</instructions>

<examples>
## Good
```go
import (
    "io"
    "os"
)
```
</examples>

<patterns>
- Replace deprecated `io/ioutil` imports with `io` and `os`
- Remove imported packages that are never referenced in code
- Replace overlapping imports with the smaller more specific package
- Replace package imports used for a single function available elsewhere
</patterns>
