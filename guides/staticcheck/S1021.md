# staticcheck: S1021

<instructions>
Detects code that declares a variable with `var` and then immediately assigns a composite literal. The declaration and initialization can be combined into a single `:=` statement with the composite literal.

Use short variable declaration with the composite literal directly.
</instructions>

<examples>
## Bad
```go
var m map[string]int
m = map[string]int{}
```

## Good
```go
m := map[string]int{}
```
</examples>

<patterns>
- Declaring a variable then assigning an empty composite literal
- var + assignment where short declaration suffices
- Separate declaration and initialization of slices, maps, or structs
- Using var for types that can be inferred from the literal
</patterns>
