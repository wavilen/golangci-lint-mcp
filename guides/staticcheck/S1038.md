# staticcheck: S1038

<instructions>
Detects unnecessary formatting operations where a simpler alternative exists. This includes using `fmt.Sprintf` for string concatenation or formatting when direct operations would suffice.

Use string concatenation or simpler formatting approaches.
</instructions>

<examples>
## Bad
```go
msg := fmt.Sprintf("Hello, %s!", name)
// when name is guaranteed to be a simple string
```

## Good
```go
msg := "Hello, " + name + "!"
```
</examples>

<patterns>
- Using fmt.Sprintf for simple string concatenation
- Formatting with %s for a single string substitution
- Using fmt.Sprintf where strings.Replace would be clearer
- Over-formatting when direct concatenation is simpler
</patterns>
