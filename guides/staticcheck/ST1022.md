# staticcheck: ST1022

<instructions>
Detects unnecessary statements or expressions in function bodies. This includes dead code after return statements, unreachable code, or expressions that have no side effects.

Remove the dead or unreachable code.
</instructions>

<examples>
## Bad
```go
func done() error {
    return nil
    log.Println("never reached") // dead code
}
```

## Good
```go
func done() error {
    return nil
}
```
</examples>

<patterns>
- Remove code after `return`, `panic`, or `os.Exit` that is never reached
- Remove unreachable case branches in switch statements
- Remove dead code in conditional branches that are always true or always false
- Remove unreachable defer statements
</patterns>
