# staticcheck: S1023

<instructions>
Detects hardcoded paths like `"/dev/null"` or `"NUL"` instead of using `os.DevNull`. The `os.DevNull` constant is portable across operating systems and makes the intent explicit.

Replace the hardcoded device path with `os.DevNull`.
</instructions>

<examples>
## Good
```go
f, err := os.Open(os.DevNull)
```
</examples>

<patterns>
- Replace hardcoded `/dev/null` with `os.DevNull` for cross-platform support
- Replace `NUL` with `os.DevNull` for portability
- Replace hardcoded null device paths in test fixtures with `os.DevNull`
- Replace `/dev/null` hardcoded paths with `os.DevNull` when discarding output
</patterns>
