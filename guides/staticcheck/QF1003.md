# staticcheck: QF1003

<instructions>
Detects uses of `strings.Replace` with the count argument set to `-1` (replace all). Since Go 1.12, `strings.ReplaceAll` is the clearer equivalent. The same applies to `bytes.Replace` → `bytes.ReplaceAll`.

Use `strings.ReplaceAll` or `bytes.ReplaceAll` instead.
</instructions>

<examples>
## Good
```go
result := strings.ReplaceAll(s, "old", "new")
```
</examples>

<patterns>
- Replace `strings.Replace(s, old, new, -1)` with `strings.ReplaceAll`
- Replace `bytes.Replace(b, old, new, -1)` with `bytes.ReplaceAll`
- Use `ReplaceAll` when replacing all occurrences
- Prefer the more descriptive `ReplaceAll` API over `Replace` with `-1`
</patterns>
