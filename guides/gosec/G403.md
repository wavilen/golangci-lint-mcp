# G403

<instructions>
Detects RSA keys shorter than 2048 bits. Short RSA keys (512, 1024 bits) can be factored by attackers, breaking encryption and signature security.

Generate RSA keys with at least 2048 bits (prefer 4096 for long-term security). Use `rsa.GenerateKey(rand.Reader, 2048)` or stronger.
</instructions>

<examples>
## Bad
```go
key, _ := rsa.GenerateKey(rand.Reader, 1024)
```

## Good
```go
key, err := rsa.GenerateKey(rand.Reader, 4096)
if err != nil {
    return err
}
```
</examples>

<patterns>
- RSA key generation with less than 2048 bits
- Hardcoded PEM blocks containing weak RSA keys
- Key size parameters from configuration allowing values below 2048
</patterns>

<related>
G401, G402, G407
</related>
