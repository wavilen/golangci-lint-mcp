# staticcheck: SA1026

<instructions>
Detects `net.Listen` called with a service name (e.g., `"http"`, `"smtp"`) instead of a numeric port. Service names require a lookup in `/etc/services` and may not exist on all systems. Use a numeric port string like `":8080"` for reliability.
</instructions>

<examples>
## Bad
```go
ln, err := net.Listen("tcp", ":http") // relies on /etc/services lookup
```

## Good
```go
ln, err := net.Listen("tcp", ":8080")
```
</examples>

<patterns>
- Replace service names in `net.Listen` with numeric port strings
- Use numeric ports for portability
- Replace hardcoded service names with numeric ports to avoid unexpected port mappings
</patterns>
