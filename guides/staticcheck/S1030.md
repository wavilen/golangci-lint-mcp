# staticcheck: S1030

<instructions>
Detects uses of `fmt.Sprintf` to convert numeric values to strings when `strconv` functions are more efficient and clearer. `strconv.Itoa`, `strconv.FormatFloat`, and `strconv.FormatInt` avoid the overhead of format parsing.

Use the appropriate `strconv` function instead of `fmt.Sprintf`.
</instructions>

<examples>
## Good
```go
s := strconv.Itoa(count)
```
</examples>

<patterns>
- Replace `fmt.Sprintf` with `strconv.Itoa` for int-to-string conversion
- Replace `fmt.Sprintf` with `strconv.FormatFloat` for float formatting
- Replace `fmt.Sprintf` with `strconv.FormatInt` for hex formatting
- Use `strconv` functions for any simple numeric-to-string conversion
</patterns>
