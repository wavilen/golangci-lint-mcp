# staticcheck: S1029

<instructions>
Detects unnecessary formatting operations where the value is already suitable for direct use. This includes using `fmt.Sprintf` to build a string that could be constructed more simply.

Simplify by removing the formatting layer or using a direct conversion.
</instructions>

<examples>
## Good
```go
msg := "error: " + err.Error()
// or better, use errors.Wrap for wrapping:
// return errors.Wrap(err, "error")
```
</examples>

<patterns>
- Replace `fmt.Sprintf` concatenation with `+` operator for simple string joining
- Replace `fmt.Sprintf` with direct string use when already a string
- Remove formatting of single values that do not need formatting
- Simplify over-engineered `Sprintf` string construction with direct operations
</patterns>
