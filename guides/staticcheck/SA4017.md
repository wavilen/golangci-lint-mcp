# staticcheck: SA4017

<instructions>
Detects function calls whose return values are discarded when they shouldn't be. This includes calls to functions that return errors, functions with side effects that communicate results via return values, and pure functions whose entire purpose is the return value.

Assign the return value to a variable and handle it, or use the blank identifier `_` only when the discard is intentional and safe.
</instructions>

<examples>
## Good
```go
err := os.Remove("temp.txt")
if err != nil && !os.IsNotExist(err) {
    log.Printf("failed to remove temp file: %v", err)
}
```
</examples>

<patterns>
- Handle error returns from I/O operations instead of discarding them
- Assign return values from `fmt.Fprintf`, `io.Copy`, etc. to check for errors
- Use or store return values from pure functions like `strings.Contains`
- Check return values from `http.ResponseWriter.Write` for errors
</patterns>
