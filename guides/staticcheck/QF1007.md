# staticcheck: QF1007

<instructions>
Detects uses of `fmt.Sprintf("%s", x)` where `x.String()` or a direct string method call would suffice. If the type has a `String()` method, calling it directly avoids the overhead of format parsing.

Use the `String()` method directly instead of formatting with `%s`.
</instructions>

<examples>
## Bad
```go
s := fmt.Sprintf("%s", someStringer)
```

## Good
```go
s := someStringer.String()
```
</examples>

<patterns>
- fmt.Sprintf("%s", x) where x has a String() method
- fmt.Sprintf to call String() indirectly
- Using format verbs for single-value string conversion
- Unnecessary formatting layer for string-like values
</patterns>
