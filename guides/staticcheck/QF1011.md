# staticcheck: QF1011

<instructions>
Detects code that uses deprecated or older API patterns where a modern equivalent exists. This includes using `io/ioutil` (deprecated since Go 1.16), old `context` patterns, or superseded standard library functions.

Replace the deprecated API with its modern equivalent.
</instructions>

<examples>
## Good
```go
data, err := os.ReadFile("config.json")
```
</examples>

<patterns>
- Replace `io/ioutil.ReadFile` with `os.ReadFile`
- Replace `io/ioutil.WriteFile` with `os.WriteFile`
- Replace `io/ioutil.TempDir` with `os.MkdirTemp`
- Replace `io/ioutil.TempFile` with `os.CreateTemp`
</patterns>
