# staticcheck: S1007

<instructions>
Detects manual URL path construction using string concatenation, `path.Join`, or `filepath.Join` with URL components. The `url.JoinPath` function handles trailing slashes and path segments correctly for URLs.

Replace manual URL path building with `url.JoinPath(base, parts...)`.
</instructions>

<examples>
## Bad
```go
base := "https://example.com/api/"
endpoint := base + "users/" + strconv.Itoa(id)
```

## Good
```go
endpoint, err := url.JoinPath("https://example.com/api", "users", strconv.Itoa(id))
```
</examples>

<patterns>
- Concatenating URL base with path segments using +
- Using path.Join for URLs (it strips trailing slashes differently)
- Building API URLs from base + path + query manually
- Manual slash management between URL segments
</patterns>
