# staticcheck: SA1007

<instructions>
Detects URL strings that contain space characters. Spaces in URLs are invalid per RFC 3986 and will cause failures with `http.Get`, `url.Parse`, or any HTTP client call. URL-encode the space as `%20` or use `url.URL` to construct the URL properly.
</instructions>

<examples>
## Bad
```go
resp, err := http.Get("http://example.com/path with spaces")
```

## Good
```go
u := url.URL{
    Scheme: "http",
    Host:   "example.com",
    Path:   "/path with spaces",
}
resp, err := http.Get(u.String()) // automatically encodes spaces
```
</examples>

<patterns>
- Hardcoded URLs with spaces in string literals
- Concatenating URL paths with unencoded user input
- Building query strings manually instead of using `url.Values`
</patterns>
