# staticcheck: SA4014

<instructions>
Detects comparisons that are always true or always false due to the types or values involved. This includes comparing identical ranges, comparing constants with values outside their type range, or tautological conditions. These are dead code — one branch is never taken.

Fix the comparison logic, remove the dead branch, or correct the types involved.
</instructions>

<examples>
## Good
```go
x := getValue()
if x == 5 {
    slog.Info("conditional")
}
```
</examples>

<patterns>
- Remove tautological comparisons of a constant to itself or its known value
- Fix range checks like `x > 0 && x < 1` for integer types — always false
- Fix type-constrained values compared outside their valid domain
- Simplify boolean expressions that resolve to constant true/false
</patterns>
