# staticcheck: S1015

<instructions>
Detects uses of `fmt.Sprintf` to convert a simple value to a string when a direct conversion or `strconv` function would suffice. Using `fmt.Sprintf` for simple conversions adds unnecessary overhead and complexity.

Use `strconv.Itoa`, `strconv.FormatFloat`, or a direct type conversion instead.
</instructions>

<examples>
## Bad
```go
s := fmt.Sprintf("%d", 42)
```

## Good
```go
s := strconv.Itoa(42)
```
</examples>

<patterns>
- Replace `fmt.Sprintf` with `strconv.Itoa` for int-to-string conversion
- Replace `fmt.Sprintf` with direct string use when already a string
- Use `strconv` functions for simple type-to-string conversions instead of `Sprintf`
- Replace `fmt.Sprintf` with `strconv` or `string()` when possible
</patterns>
