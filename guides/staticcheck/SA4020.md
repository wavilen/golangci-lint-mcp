# staticcheck: SA4020

<instructions>
Detects unreachable cases in type switches where the case type can never match the interface value because the static type of the expression already constrains the possible runtime types. The case is dead code.

Remove the unreachable case, or fix the switch expression to use an interface type that allows the match.
</instructions>

<examples>
## Bad
```go
var w io.Writer = os.Stdout
switch v := w.(type) {
case *os.File:
    slog.Info("file")
case *bytes.Buffer:
    slog.Info("buffer") // unreachable: *bytes.Buffer never assigned to w
}
```

## Good
```go
switch v := w.(type) {
case *os.File:
    slog.Info("file")
case io.Writer:
    slog.Info("other writer", "writer", v)
}
```
</examples>

<patterns>
- Switch on interface types instead of concrete types for meaningful type switches
- Remove cases subsumed by earlier, broader cases in type switches
- Replace narrow concrete-type switches with interface-based switches
- Remove cases for types that cannot be assigned to the switched variable
</patterns>
