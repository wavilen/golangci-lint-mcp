# staticcheck: S1034

<instructions>
Detects compound boolean expressions that can be simplified using De Morgan's laws or other boolean algebra. This includes `!a && !b` → `!(a || b)`, double negations `!!x` → `x`, and other simplifiable boolean expressions.

Simplify the boolean expression to its minimal form.
</instructions>

<examples>
## Bad
```go
if !(a != b) {
    slog.Info("equal")
}
```

## Good
```go
if a == b {
    slog.Info("equal")
}
```
</examples>

<patterns>
- Double negation: !!x should be x
- !(a != b) should be a == b
- !(a == b) should be a != b
- Complex &&/|| chains that De Morgan's law can simplify
</patterns>
