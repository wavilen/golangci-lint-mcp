# staticcheck: S1037

<instructions>
Detects unnecessary else branches where the if body already handles the condition with a control flow statement. Similar to S1035 but specifically targets the structural redundancy of having an else at all.

Remove the else and de-indent the else block's code.
</instructions>

<examples>
## Good
```go
if done {
    return result
}
process()
```
</examples>

<patterns>
- Remove `else` block after `return` in the if body and de-indent
- Remove `else` block after `continue` in a loop and de-indent
- Remove `else` block after `break` in a loop and de-indent
- Remove `else` block after `panic` in the if body
</patterns>
