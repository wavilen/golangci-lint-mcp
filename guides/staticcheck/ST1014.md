# staticcheck: ST1014

<instructions>
Detects unnecessary import aliases or import groupings that could be simplified. This includes importing a package under its own name, or splitting import groups unnecessarily.

Use the default package name or consolidate import groups.
</instructions>

<examples>
## Good
```go
import (
    "fmt"
    "strings"
)
```
</examples>

<patterns>
- Remove import aliases that duplicate the package own name
- Remove unnecessary import aliases that add no clarity
- Consolidate import groups split without reason
- Use blank imports for packages imported only for side effects
</patterns>
