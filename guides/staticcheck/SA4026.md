# staticcheck: SA4026

<instructions>
Detects moderately complex dead code patterns that are not caught by simpler checks. This includes unreachable code after return/panic statements, conditions that are always true or false due to upstream logic, and branches that can never execute.

Remove the dead code or fix the control flow that makes it unreachable.
</instructions>

<examples>
## Good
```go
func process(v int) int {
    if v > 0 {
        return v
    }
    return -v
}
```
</examples>

<patterns>
- Remove statements after `return`, `panic`, or `continue` in the same block
- Remove code guarded by conditions that are always false
- Remove switch cases unreachable due to earlier exhaustive cases
- Remove unreachable error handling paths
</patterns>
