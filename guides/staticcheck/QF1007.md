# staticcheck: QF1007

<instructions>
Detects uses of `fmt.Sprintf("%s", x)` where `x.String()` or a direct string method call would suffice. If the type has a `String()` method, calling it directly avoids the overhead of format parsing.

Use the `String()` method directly instead of formatting with `%s`.
</instructions>

<examples>
## Bad
```go
s := fmt.Sprintf("%s", someStringer)
```

## Good
```go
s := someStringer.String()
```
</examples>

<patterns>
- Call `x.String()` directly instead of `fmt.Sprintf("%s", x)`
- Replace `fmt.Sprintf` with a direct `String()` call when the value implements `Stringer`
- Use direct `String()` calls instead of format verbs for single-value conversion
- Remove the unnecessary `Sprintf` formatting layer for string-like values
</patterns>
