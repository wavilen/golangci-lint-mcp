# staticcheck: S1012

<instructions>
Detects code that computes elapsed time as `time.Now().Sub(start)` instead of using the idiomatic `time.Since(start)`. The `time.Since` helper is shorter, clearer, and the standard way to measure elapsed time.

Replace `time.Now().Sub(start)` with `time.Since(start)`.
</instructions>

<examples>
## Bad
```go
start := time.Now()
doWork()
elapsed := time.Now().Sub(start)
```

## Good
```go
start := time.Now()
doWork()
elapsed := time.Since(start)
```
</examples>

<patterns>
- Replace `time.Now().Sub(start)` with `time.Since(start)` for elapsed time
- Replace manual duration subtraction with `time.Since`
- Use `time.Since` instead of `Sub` for measuring function execution time
- Replace `time.Now()` subtraction with `time.Since()` for clarity
</patterns>
