# staticcheck: SA6004

<instructions>
Detects string comparison with `==` on `[]byte` values without conversion. Comparing byte slices with string literals requires an implicit conversion each time, which allocates. For repeated comparisons, convert the byte slice to a string once.

Convert `[]byte` to `string` once before comparing, or use `bytes.Equal` for byte-slice-to-byte-slice comparisons.
</instructions>

<examples>
## Bad
```go
data := []byte("hello")
if string(data) == "hello" { // allocates on every call
    slog.Info("match")
}
```

## Good
```go
data := []byte("hello")
if bytes.Equal(data, []byte("hello")) {
    slog.Info("match")
}
```
</examples>

<patterns>
- `string(byteSlice) == "literal"` in hot paths
- Converting `[]byte` to string repeatedly in loops
- HTTP handler body comparison with string literals
- Repeated byte-to-string conversions that could be cached
</patterns>
