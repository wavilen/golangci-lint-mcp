# staticcheck: QF1012

<instructions>
Detects code patterns that can be simplified by removing unnecessary intermediate steps. This includes redundant variable assignments, unnecessary temporary values, or code that could be written more directly.

Inline or remove the unnecessary intermediate steps.
</instructions>

<examples>
## Good
```go
result := getValue()
```
</examples>

<patterns>
- Remove temporary variables that are immediately reassigned
- Eliminate unnecessary intermediate variables for simple expressions
- Remove redundant copy operations before return
- Combine two-step initialization into a single declaration
</patterns>
