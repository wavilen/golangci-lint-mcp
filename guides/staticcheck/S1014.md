# staticcheck: S1014

<instructions>
Detects `append` calls with no additional elements appended, such as `append(s)` or `append(s)`. This is a no-op — it returns the same slice without modification.

Remove the unnecessary append call entirely.
</instructions>

<examples>
## Good
```go
// Just use items directly — append with no new elements is a no-op
```
</examples>

<patterns>
- Remove `append` calls with no variadic arguments — they are no-ops
- Remove `append(s)` assignments back to the same variable
- Remove leftover debug code or partial refactoring remnants of empty `append`
- Fix accidental empty `append` left after removing elements
</patterns>
