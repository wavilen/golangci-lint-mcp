# staticcheck: S1015

<instructions>
Detects uses of `fmt.Sprintf` to convert a simple value to a string when a direct conversion or `strconv` function would suffice. Using `fmt.Sprintf` for simple conversions adds unnecessary overhead and complexity.

Use `strconv.Itoa`, `strconv.FormatFloat`, or a direct type conversion instead.
</instructions>

<examples>
## Bad
```go
s := fmt.Sprintf("%d", 42)
```

## Good
```go
s := strconv.Itoa(42)
```
</examples>

<patterns>
- Using fmt.Sprintf("%d", n) to convert int to string
- Using fmt.Sprintf("%s", s) when s is already a string
- Using fmt.Sprintf for simple type-to-string conversions
- Using fmt.Sprintf("%v", x) where strconv or string() works
</patterns>
