# staticcheck: ST1010

<instructions>
Detects unnecessary string conversions where the value is already a string or can be used as one directly. This includes `string(b)` on byte slices where `string(b)` is the only operation, or converting a `fmt.Stringer` result that is already a string.

Remove the unnecessary string conversion or use the value directly.
</instructions>

<examples>
## Good
```go
name := getString()
```
</examples>

<patterns>
- Remove redundant string conversions on values already of type string
- Remove unnecessary string wrapping on values that are already strings
- Remove unnecessary byte-to-string conversions for equality comparison
- Remove redundant string wrapping on `fmt.Sprintf` results
</patterns>
