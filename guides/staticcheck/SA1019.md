# staticcheck: SA1019

<instructions>
Detects use of deprecated standard library functions, types, or methods. Deprecated APIs may be removed in future Go versions, have known bugs, or have been superseded by better alternatives. The Go stdlib marks deprecated items with `// Deprecated:` comments.

Replace the deprecated API with its documented replacement. Common examples: `io/ioutil` → `io`/`os`, `interface{}` → `any`, `(*os.File).Readdir` → `ReadDir`.
</instructions>

<examples>
## Bad
```go
data, err := ioutil.ReadFile("config.json") // ioutil deprecated since Go 1.16
```

## Good
```go
data, err := os.ReadFile("config.json")
```
</examples>

<patterns>
- Replace `io/ioutil` functions with `os` and `io` equivalents
- Replace `interface{}` with `any` (deprecated alias since Go 1.18+)
- Replace deprecated `Readdir` with `ReadDir`
- Replace deprecated `Readdirnames` with `ReadDir`
</patterns>
